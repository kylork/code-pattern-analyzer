Code Pattern Analyzer API

Welcome to the Code Pattern Analyzer API. This service provides endpoints for analyzing source code to identify patterns, design patterns, and code smells.

Interactive API Documentation

Key Endpoints

GET /patterns

List all available patterns that can be detected.

GET /categories

List all available pattern categories and the patterns in each category.

POST /analyze

Analyze code content submitted in the request body.

POST /upload

Upload and analyze a single file.

GET /analysis/{analysis_id}

Get the results of a previous analysis.

GET /analysis/{analysis_id}/html

Get an HTML report for a previous analysis.

POST /projects

Create a new project for organizing multiple files.

POST /projects/{project_id}/files

Upload a file to a project.

POST /projects/{project_id}/analyze

Analyze all files in a project.

Implementation Options

The analyzer supports two implementation modes:

  • Real Implementation: Uses tree-sitter for accurate AST-based analysis (default)
  • Mock Implementation: Uses regex-based matching for quick demonstration

Most endpoints accept a use_mock parameter to toggle between these modes.
